## E-WordChanger 2005

**種別**: シェアウェア（販売終了 / 開発停止）

英語版アプリケーションのメニューやダイアログ内の文字列を **日本語に変換** するソフトウェアです（[E-WordChanger 2000](../ewordchanger/) の後継）。

- フォント変更により、日本語表示できない英語版ソフトも日本語表示が可能
- 日本語化時の文字切れをある程度修正
- アプリごとに辞書を用意して日本語化。辞書は自作可能
- **2000 → 2005 の主な変更点**: 従来は英語が表示されてから日本語に再設定していたのを、**表示される前に変換**するよう改良
- 辞書の作成: 設定画面でターゲットを追加 → 再起動してウィンドウを開くと `\Program Files\EpSoft\E-WordChanger2005\Data\` に辞書が生成されます

**対応OS**: Windows XP

## E-IPConfig Ver1.20

**種別**: フリーソフト

Windows 2000 付属のコマンドプロンプト用 `ipconfig.exe` の **GUI版** です。
`release` および `renew` 機能をサポートしています。

**対応OS**: Windows 2000 / XP
