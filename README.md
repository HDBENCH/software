# HDBENCH.NET Software

Windows向けのベンチマーク／キャプチャ／ユーティリティを公開しています。
各アプリは右の **[Releases](https://github.com/HDBENCH/software/releases)** からZIPをダウンロードできます。
アプリ名をクリックすると、それぞれの詳細（機能・対応OS）を確認できます。

## ベンチマーク
| アプリ | 種別 | 概要 |
|--------|------|------|
| **[HDBENCH](hdbench/)** | フリーソフト | PC性能の総合ベンチマーク（CPU/メモリ/ディスク/ビデオ） |
| **[FDBENCH](fdbench/)** | フリーソフト | ストレージ（HDD/USB/フラッシュ）性能ベンチマーク |
| **[DDBENCH](ddbench/)** | フリーソフト | DirectDrawの描画性能ベンチマーク |

## キャプチャ／録画

|--------|------|------|
| **[DxRec2](dxrec2/)** | シェアウェア（販売終了） | DirectX/OpenGLアプリのFPS表示・動画録画 |
| **[DxRec](dxrec/)** | シェアウェア（販売終了） | FPS表示・ベンチマーク・録画（DxRec2の前世代） |
