# E-WordChanger 2000

**種別**: シェアウェア（販売終了）

## 概要
英語版アプリケーションのメニューやダイアログ内の文字列を **日本語に変換** するソフトウェアです。

## 特徴
- 通常は日本語表示できない英語版ソフトも、**フォント変更**により日本語表示が可能
- 日本語化した際の**文字切れ**もある程度修正できる
- アプリケーションごとに**辞書**を用意することで、ほとんどの項目を日本語化可能
- 辞書は**自作**でき、**日本語→英語**への変換にも対応

## 対応OS
- Windows 98 / 2000 / XP（Windows 95 / NT4.0 は未確認）
