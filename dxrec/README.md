# DxRec

**種別**: シェアウェア（販売終了）

> 後継の **[DxRec2](../dxrec2/)** の公開に伴い、DxRec（初代）のサポートは終了しています。新規に使う場合は DxRec2 を推奨します。

## 概要
DirectX / OpenGL を利用したアプリケーション（ゲーム等）に、**フレームレート表示・ベンチマーク機能** を追加します。スクリーンショット保存、AVI動画録画機能もあります。

## 特徴
- ゲーム側に自動的に動画処理を追加し、FPS表示や動画録画が可能
- 参照型AVI保存で数時間レベルの長時間録画
- ビデオカードのバッファから直接取得する高速録画（DirectX8/9 はハードウェア縮小対応）
- Windows Vista のオーディオストリームキャプチャ対応（5.1chサラウンド／USB出力の音声も取得可）
- 録画したAVIは Windows Media Player で再生可能

## 対応形式
- DirectX 1 / 2 / 3 / 5 / 6 / 7 / 8 / 9 / 10
- OpenGL

## 対応OS
- Windows 2000 SP4 以降
- Windows XP SP2 以降 / XP x64（Wow64）
- Windows Vista x86 / x64（Wow64）
- ※ Windows 9x は非対応
