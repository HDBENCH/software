# FDBENCH

**種別**: フリーソフト

## 概要
HDD・フラッシュディスク・USBフラッシュドライブの性能を測定するベンチマークソフトです。

計測項目：

| 項目 | 内容 |
|------|------|
| Read / Write | シーケンシャル リード・ライト速度（KByte/sec） |
| RandomRead / RandomWrite | ランダム リード・ライト速度（KByte/sec） |
| Copy | Read / Write / Delete の回数/分 または速度（KByte/秒） |

## 対応OS
- Windows 95 / 98 / NT4.0 / 2000 / XP
