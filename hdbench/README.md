# HDBENCH

**種別**: フリーソフト

## 概要
パソコンの **CPU・DISK・VIDEO** の性能を測定する総合ベンチマークソフトです。
HDD以外の測定は **Dual CPU** に対応しています。

## 対応OS
- Windows 95 / 98 / NT4.0 / 2000 / XP

## ダウンロード
[Releases](https://github.com/HDBENCH/software/releases) からZIPをダウンロードしてください。
