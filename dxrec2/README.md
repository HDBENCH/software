# DxRec2

**種別**: シェアウェア（販売終了）

## 概要
DirectX / OpenGL を利用したゲーム等に、**フレームレート表示・スクリーンショット保存・AVI動画録画** 機能を追加するソフトウェアです。DxRec（初代）の後継として、速度と機能を大幅に強化しています。

## 特徴
- **世界最速を目指した新規開発**。フォント描画を廃し、SSE専用版を用意
- **マルチコア対応**（Quadコアまで自動でキャプチャを割り振り）
- **64ビットアプリケーション／ゲームに完全対応**
- **自動フレームスキップ**でメモリ消費を抑制
- **スクリーンショット**は BMP / JPEG / GIF / PNG / TIFF に対応（同時保存・サイズ変更可）
- **参照型AVI保存**により数時間レベルの長時間録画が可能
- ビデオカードのバッファから直接取得する高速録画。DirectX8/9 ではハードウェア縮小を使用
- Windows Vista の**オーディオストリームキャプチャ**に対応（5.1chサラウンド／USBスピーカー・ヘッドフォン出力の音声も取得可）
- 録画したAVIは Windows Media Player で再生可能

## 対応形式
- DirectX 1 / 2 / 3 / 5 / 6 / 7 / 8 / 9 / 10
- OpenGL
